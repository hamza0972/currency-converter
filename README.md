# 💱 Real-Time Currency Converter (Streamlit)

## 📌 Project Overview
This is a simple real-time currency converter web application built using **Streamlit** and a **REST API**. It allows users to convert between multiple world currencies using live exchange rates.

---

## 🚀 Features
- Real-time currency conversion
- Multiple global currencies support
- Simple and interactive UI
- Lightweight and fast Streamlit app
- API-based live exchange rates

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Requests Library
- ExchangeRate API

---

## 📷 How it works
1. User selects "From Currency"
2. User selects "To Currency"
3. Enters amount
4. App fetches live exchange rate from API
5. Displays converted value instantly
